# JAVA Practice

This repository contains my Java practice codes.



## 📁 File Overview

| File Name        | Description |
|------------------|-------------|
| **Main.java** | A simple Hello World program that prints "Hello JAVA" to the console. | 
| **Main_1.java** | This program displays product information. It prints the ID, title, price, description, and category sequentially as output. | 
| **Main_2.java** | perfect product with best image quality. | 
| **Main_3.java** | This program uses the Scanner class to take user input for product information. The user enters the product id, title, price, description, and category. After taking all inputs, the program displays the product details using System.out.println(). | 


## How to run

1. Make sure you have Java installed.
2. Compile: `javac Main.java`
3. Run: `java Main`

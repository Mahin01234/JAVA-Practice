## 📁 About the Repository

This collection of Java programs is designed to help beginners practice core concepts. The code progresses from simple “Hello, World!” programs to more advanced OOP examples, offering a solid foundation in Java programming.

---

## 📜 File Descriptions

### Basics & Syntax
| File Name | Description |
|-----------|-------------|
| `Main.java` | Prints “Hello JAVA” to the console – the classic starting point. |
| `Variable.java` | Declares two integers, calculates their sum, and prints the result. |
| `Variables_1.java` | Demonstrates multiple data types (`String`, `int`, `float`, `char`) using student details. |
| `Variables_2.java` | Calculates the area of a rectangle given length and width. |
| `Print_Variables.java` | Shows string concatenation by combining `firstName` and `lastName`. |
| `Data_type.java` | Illustrates all 8 basic Java data types (`byte`, `short`, `int`, `long`, `float`, `double`, `char`, `boolean`). |
| `DataTypesComplete.java` | Covers basic and non‑basic types (`String`, arrays, `var`) and includes a shopping cart example. |
| `JavaSyntax.java` | Highlights essential Java syntax rules. |
| `Java_Variables.java` | Another variables example for extra practice. |

### Input / Output
| File Name | Description |
|-----------|-------------|
| `Main_1.java` | Displays product information (ID, title, price, description, category). |
| `Main_2.java` | Extends product information with image quality details. |
| `Main_3.java` | Uses `Scanner` to collect product information from the user and then displays it. |
| `Print_Numbers.java` | Prints numbers in a formatted way. |
| `Public_Details.java` | Shows how to manage and display public class fields. |

### Patterns
| File Name | Description |
|-----------|-------------|
| `Main_4.java` | Prints a right‑angled triangle of stars (1 to 5 stars) using only `System.out.println()`. |
| `Main_5.java` | Prints an inverted right‑angled triangle of stars (5 to 1 stars). |
| `Main_6.java` | Creates a centered pyramid pattern using stars. |

### String Operations
| File Name | Description |

| `CharAt.java` | Demonstrates the `charAt()` method to extract characters from a string. |
| `Equals.java` | Compares strings using the `equals()` method. |
| `Index_of.java` | Finds the index of a character or substring in a string. |
| `Length.java` | Shows how to get the length of a string. |

### Control Flow
| File Name | Description |
|-----------|-------------|
| `IF_ELSE.java` | Basic `if‑else` conditional logic. |
| `Else_IF.java` | Demonstrates `else if` chains for multiple conditions. |
| `ForLoop.java` | Simple `for` loop example. |
| `BreakExample.java` | Uses `break` to exit a loop early. |

| `Book_Details.java` | Extends `Book` with more detailed book information. |

### Prerequisites
- **Java Development Kit (JDK)** – version 8 or later.  
  Download it from [Oracle’s official website](https://www.oracle.com/java/technologies/javase-downloads.html) or use a package manager like `apt` (Linux) or `brew` (macOS).
- A **text editor** or **IDE** (IntelliJ IDEA, Eclipse, VS Code, etc.).
